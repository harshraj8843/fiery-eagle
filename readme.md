## üèÅ Getting Started <a name = "getting_started"></a>

- clone project
  ```
  git clone https://github.com/novicexp/fiery-eagle
  ```
  ```
  cd fiery-eagle/
  ```
- install requirements
  ```
  pip install -r requirements.txt
  ```
- migrate
  ```
  python manage.py migrate
  ```
- runserver
  ```
  python manage.py runserver
  ```
